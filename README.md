# Spring Boot External API Clients

This project demonstrates how to call external APIs in Spring Boot using different HTTP clients, along with basic implementation.

## Covered Topics

- Why external APIs are needed in Spring Boot
- RestTemplate (synchronous, legacy)
- WebClient (reactive, modern)
- HttpExchange (low-level control)
- RestClient (simplified modern approach)

## Tech Stack

- Java
- Spring Boot
- Spring Web
- Spring WebFlux

## Run

1. Clone the repo
2. Run the application
3. Test endpoints using Postman or browser
